# 实验四 实验报告（8255并行接口实验）

1. 实验目的

- 学习并掌握8255的工作方式及其应用。
- 掌握8255典型应用电路的接法。
- 掌握程序固化及脱机运行程序的方法（可选）。

2. 实验内容

- 基本输入输出实验：端口A作为输出，端口B作为输入；拨动开关变化通过8255实时反映到数据灯。

3. 实验步骤

1) 按实验接线图连接8255与拨动开关、数据灯。
2) 编写汇编程序（A82551.ASM），装入并运行。
3) 拨动开关，观察LED显示；验证响应即时性和正确性。
4) （可选）将程序固化并验证断电重启后能否脱机运行。

4. 程序清单

见预习报告中的 `A82551.ASM` 示例程序。

5. 实验分析

- 控制字配置：通过对控制端口写入控制字，使端口A为输出、端口B为输入并选择方式0，这是实现简单并行输入/输出的常用配置。
- 实时性：在方式0下，端口B为简单输入，读入后直接送出到端口A，响应时间主要受CPU轮询速度及I/O延时影响；若需更高效可用中断或DMA（若支持）。
- 固化验证：固化程序可用来验证在断电/复位后系统能否脱机运行，实验中应注意JDBG跳线的RUN/DBG配置以切换联机与脱机模式。
- 调试要点：若LED不响应，应检查基址设置、控制字是否正确写入、以及物理连线（端口与开关/灯）与电源信号是否正常。
