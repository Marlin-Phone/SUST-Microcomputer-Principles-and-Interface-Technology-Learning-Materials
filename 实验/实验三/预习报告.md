# 实验三 预习报告（静态存储器扩展实验）

1. 实验目的

- 了解存储器扩展的方法与存储器的读/写操作。
- 掌握CPU对16位存储器的访问方式（字存取与地址步进）。

2. 实验内容

- 按规则字写存储器，编写并调试程序，将16个字（0000H~000FH）写入SRAM从8000:0000H起的连续空间，并用系统命令查看验证。

3. 实验步骤

1) 按实验接线图连接SRAM与系统总线（见课程手册图4.1.4）。
2) 编写、汇编并链接程序（例程名MEM.ASM），将生成的可执行装入系统。
3) 运行程序，待程序停止。
4) 使用D命令查看写入的数据：例如输入 `D8000:0000` 并回车，检查显示是否为 0000、0001、...、000F。

4. 程序清单

; 文件名: MEM.ASM
; 功能描述: 扩展存储器实验, 访问16位存储器, 将16个数写入从8000:0000H开始的连续地址单元

SSTACK SEGMENT STACK
    DW 32 DUP(?)
SSTACK ENDS

CODE SEGMENT
START PROC FAR
    ASSUME CS:CODE
    MOV AX, 8000H    ; 存储器扩展空间段地址
    MOV DS, AX

    MOV SI, 0000H    ; 数据首地址 (段内偏移)
    MOV CX, 0010H    ; 要写入的字数 16
    MOV AX, 0000H
AA1:
    MOV [SI], AX     ; 将 AX 写入 DS:SI（16位存储器，步进两字节）
    INC AX
    ADD SI, 2        ; 下一个字地址（因为是16位字）
    LOOP AA1

    MOV AX, 4C00H
    INT 21H          ; 程序终止
START ENDP
CODE ENDS
END START
