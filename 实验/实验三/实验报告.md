# 实验三 实验报告（静态存储器扩展实验）

1. 实验目的

- 了解存储器扩展的方法与存储器的读/写操作。
- 掌握CPU对16位存储器的访问方式（字存取与地址步进）。

2. 实验内容

- 将0000H~000FH共16个数写入SRAM从8000:0000H起的一段空间，运行并用系统命令查看验证写入数据。

3. 实验步骤

1) 按图4.1.4接线并检查连接无误。
2) 将 `MEM.ASM` 程序汇编、链接并装入系统。
3) 运行程序，程序运行结束后使用 `D8000:0000` 命令查看内存内容。

4. 程序清单

见预习报告中的 `MEM.ASM`。

5. 实验分析

- 结果：通过 `D8000:0000` 可以看到从0000到000F的16个字按顺序存放，验证写入正确。
- 地址控制：由于访问为16位字（字长2字节），在段内偏移地址上每写一字需增加2（程序中用 `ADD SI,2` 实现），这是16位存储器访问的关键。
- 紧要点：在硬件上要注意数据总线的高低位选通信号（BHE# / A0等）以保证CPU与16位SRAM之间的数据方向和选通正确。
- 调试提示：若读到的数据不正确，先检查片选和读/写信号（WR#/RD#）的连接，再检查段地址DS是否正确设置为扩展存储器段（如8000H）。
