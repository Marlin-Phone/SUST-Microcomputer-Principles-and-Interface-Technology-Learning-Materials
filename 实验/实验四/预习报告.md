# 实验四 预习报告（8255并行接口实验）

1. 实验目的

- 学习并掌握8255的工作方式及其应用。
- 掌握8255典型应用电路的接法。
- 掌握程序固化及脱机运行程序的方法（可选）。

2. 实验内容

- 基本输入输出实验：将8255的端口A配置为输出，端口B配置为输入；用拨动开关驱动端口B，端口A连接数据灯，要求开关拨动时数据灯实时反映变化。

3. 实验步骤

1) 按实验接线图（图4.3.4）接线：拨动开关接入端口B，端口A接数据灯。
2) 编写并汇编程序（例程名A82551.ASM），装入系统。
3) 运行程序，拨动开关并观察LED显示，验证输入输出功能。
4) （可选）固化程序：在系统中选择“固化程序”，将代码写入系统存储器并验证脱机运行。

4. 程序清单（示例）

; 文件名: A82551.ASM
; 功能描述: 将8255的端口A设为输出，端口B设为输入；实时将B端口数据送到A端口显示

PORTA  EQU 0378H      ; 例：8255 基址（根据实验板确定）
PORTB  EQU PORTA+1
PORTC  EQU PORTA+2
CTRLP  EQU PORTA+3

START:
    MOV DX, CTRLP
    MOV AL, 10010000B   ; 控制字：D7=1(模式设置), 设置端口A为输出，端口B为输入，方式0（示例控制字）
    OUT DX, AL

MAIN_LOOP:
    MOV DX, PORTB
    IN AL, DX
    MOV DX, PORTA
    OUT DX, AL
    JMP MAIN_LOOP

; 说明：具体基址及控制字视实验板手册和接线而定，上述为教学示例。
